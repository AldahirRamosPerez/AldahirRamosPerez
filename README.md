### 📜 My Story

My journey into tech started with a simple curiosity about how software could solve everyday problems. I quickly dove into Python and R, fascinated by the power of data to tell stories and predict behaviors. That passion led me to master web development with JavaScript and Node.js, where I learned to build the applications that bring those insights to life.

Along the way, I realized that true value lies in automation. I spent time creating workflows that save hours of repetitive tasks, like a system to manage calendars and Android devices using Samsung Routines. Recently, I've immersed myself in the world of **large language models** and **retrieval-augmented generation (RAG)**, building intelligent agents powered by **DeepSeek** that can query, reason, and respond with context-aware precision.

After collaborating on various data and web projects, the moment to build something of my own arrived. My goal now is to create independent digital products that fuse data science, clean web interfaces, and cutting-edge AI. I want to share that process — the wins and the mistakes — to show that a solid bridge can be built between analysis, user experience, and the latest in generative AI. The journey is just getting started.
